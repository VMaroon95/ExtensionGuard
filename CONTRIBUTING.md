## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your feature: `git checkout -b feature/my-feature`
3. **Make your changes** and test them
4. **Commit** with a clear message: `git commit -m "Add: description of change"`
5. **Push** and open a **Pull Request**

## Ideas for Contributions

- Add more permissions to the database (`backend/permissions_db.py`)
- Improve Chrome Web Store scraping accuracy
- Add browser extension version (scan directly from your browser)
- Improve UI/UX
- Add tests
- Documentation improvements

## Code Style

- Python: Follow PEP 8
- JavaScript: Use `const`/`let`, no `var`
- Keep it simple and readable

## Reporting Issues

Open a GitHub issue with:
- What you expected
- What actually happened
- Steps to reproduce
